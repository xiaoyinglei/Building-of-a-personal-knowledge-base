# Agentic RAG Overview

## Reliability First

Reliable retrieval is more important than fluent synthesis. The system should cite chunks, expose conflicts, and downgrade confidence when support is weak.

## Dual Runtime

Fast Path handles direct lookup and single-document questions. Deep Path handles comparison, synthesis, and research tasks with bounded iterative retrieval.

## Preservation

Useful answers can become topic pages, comparison pages, or open-question pages after user approval.
