# Memory Scope

## Working Memory

Session-local sub-questions and evidence matrices live in `SessionRuntime` and are not durable by default.

## Semantic Memory

Approved knowledge artifacts become durable retrieval assets.

## Operational Memory

Telemetry events track retrieval branches, escalations, claim-citation failures, and preservation activity.

## Episodic Memory

Session summaries are deferred beyond the v1 baseline unless promoted into approved artifacts.
